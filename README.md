# v2.0.0

Authors: Crispin, J.J., Bateman, A.H., Struzik, Z., Campbell, A., Voyagaki, E., Mylonakis, G., Vardanega, P.J.

Released: January 2026

This Technical Appendix and associated python script details the MSD method to calculate wall displacements using the mobilisable strength design (MSD) method. The first stage details the rigid wall rotation of a cantilever retaining wall, followed by muliple stages of a propped excavation.

v2.0.0 of this code has been used in the following publication:
Hua, Y., Struzik, Z., Bateman, A.H., Crispin, J.J., Huang, H., Zhang, D., Mylonakis, G.E. & Vardanega, P.J. (2026).
Assessing the effectiveness of the mobilizable strength design method for prediction of retaining wall movements using a database.
Submitted.

v1.0.0 of this code has been used in the following publication:
Crispin, J.J., Bateman, A.H., Voyagaki, E., Campbell, A., Mylonakis, G., Bolton, M.D. & Vardanega, P.J. (2024). 
MSD applied to the construction of the British Library basement: a multistage excavation in London Clay. 
Canadian Geotechnical Journal, 61(3):596-603.  https://doi.org/10.1139/cgj-2023-0238
